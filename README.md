============

jiaxf test nw app

package.json生成
====
```
	npm init
```

npm install modules
====

```
	npm install [package name]
```

_--save参数表示将该模块写入dependencies属性，_

_--save-dev表示将该模块写入devDependencies属性_

Package & Installer
====

- Build the release application and package to installers:

```
	grunt app package
```
